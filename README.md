### Ejemplo de directiva para un switch usando: ###

* Express
* AngularJS
* Node.js
* GruntJS

### Para comenzar ###

Clonar el ropositorio

>
    $ git clone https://github.com/adesiseric/switch-example-with-angularjs.git

Instalar dependencias de node
>
    $ sudo npm install

Instalar dependencias de bower
>
    $ bower install

Correr el proyecto
>
    $ grunt
